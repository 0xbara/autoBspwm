# Enviroment Hotkeys
### Independent

<kbd>â–</kbd> + <kbd>Enter</kbd> : Open terminal (kitty).    
<kbd>â–</kbd> + <kbd>1,2,3,4...</kbd> : Focus to the given desktop.    
<kbd>â–</kbd> + <kbd>Shift</kbd> + <kbd>1,2,3,4...</kbd> : Send to the given desktop.    
<kbd>â–</kbd> + <kbd>D</kbd> : Program launcher.    
<kbd>â–</kbd> + <kbd>ESC</kbd> : Make sxhkd reload its configuration files.    

### Bspwm

<kbd>â–</kbd> + <kbd>Alt</kbd> + <kbd>(q,r)</kbd> : Quit or Restart bspwm.    
<kbd>â–</kbd> + <kbd>W</kbd> : Close and Kill.    
<kbd>â–</kbd> + <kbd>M</kbd> : Alternate between the Tiled and Monocle layout    
<kbd>â–</kbd> + <kbd>Y</kbd> : Send the newest marked Node to the newest preselected Node.    
<kbd>â–</kbd> + <kbd>G</kbd> : Swap the current Node and the biggest window.

<kbd>â–</kbd>s + <kbd>Ctrl</kbd> + <kbd>(1-9)</kbd> : Preselect the ratio.    
<kbd>â–</kbd> + <kbd>Ctrl</kbd> + <kbd>Space</kbd> : Cancel the preselection for the focused Node.    
<kbd>â–</kbd> + <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>Space</kbd> : Cancel the preselection for the focused desktop.

# Terminal Hotkeys

<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Enter</kbd> : Open Terminal window.    
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>W</kbd> : Close Terminal window.    
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd> : Change window disposition.    
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>T</kbd> : Open terminal tab.    
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>(ğŸ  ğŸ ‘ ğŸ ’ ğŸ “)</kbd> : Change Terminal tab.    
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>T</kbd> : Change tab name.    
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>(, or .)</kbd> : Change tab order.    
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd> : Resize window.    
<kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>Left Click</kbd> : Select text.    
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>U</kbd> : Open hex help.    
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd> : Look for URL's.
